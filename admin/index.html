Editing Tips

  • Single source of truth: site-wide values come from admin/config.json.
  • Feature Toggles: use the Site Config → Feature Toggles switches to show/hide major sections.
  • Safe workflow: edit, save, then verify the affected page in the website preview.
  • Media uploads: use the collection editors or media library to upload and select files.
  • Need exact mapping? see the Feature Matrix in README.md.
